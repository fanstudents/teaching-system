FAQ

常見問題 (20 題)

關於 Claude AI 企業導入最常被問到的問題。

Claude 是什麼？跟 ChatGPT 有什麼不同？
Claude 是 Anthropic 開發的 AI 助手。與 ChatGPT 相比，Claude 擅長長文分析（1M context）、精準推理與程式設計。ChatGPT 生態系最廣泛，適合通用場景。建議企業根據需求選擇或採用多模型策略。
Claude 和 Gemini 哪個適合企業用？
如果企業已重度使用 Google Workspace（Gmail、Drive、Docs），Gemini 整合最佳。若需要精準推理、長篇合約分析或程式開發，Claude 是更好的選擇。兩者皆支援 1M+ token context。
企業導入 Claude 需要多久？
一般評估+培訓約 2-4 週。若需 API 串接或系統整合，視規模約 1-3 個月。AI Lobster 方案提供全程顧問支援。
Claude 的企業資料安全嗎？
Enterprise 方案不使用客戶資料訓練模型，支援 SOC 2、HIPAA 合規，具備 SSO/SCIM 整合與完整審計日誌。詳見 Anthropic Trust Center。
什麼是 Computer Use？
Computer Use 讓 Claude 像人一樣操作電腦螢幕 — 點擊按鈕、填寫表單、切換應用程式，完成跨軟體的自動化任務。適合 RPA 場景。
什麼是 MCP (Model Context Protocol)？
MCP 是 Anthropic 推出的開放協定（官方站），讓 Claude 可以直接連接企業工具如 Google Drive、Slack、GitHub 等，實現資料即時存取。
Claude Cowork 是什麼？
Cowork 是 2026 年推出的 GUI 代理產品，可建立角色化、持久性的 AI 代理，自動執行跨應用程式的複雜工作流。例如自動整理會議筆記並發送 follow-up email。
什麼是 Prompt 工程？為什麼重要？
Prompt 工程是設計有效指令讓 AI 產出精準結果的技巧。好的 Prompt 可以讓 AI 輸出品質提升 5-10 倍，是企業 AI 導入最重要的基礎能力。
非技術人員也能學會用 Claude 嗎？
完全可以。我們的培訓從零基礎起步，第一階段不需要任何程式能力。行銷、HR、PM 等非技術部門是最大受益者。
Claude 支援中文嗎？
支援。Claude 具備優秀的多語言能力，包含繁體中文、簡體中文、日文、韓文等，對中文的理解與生成品質持續提升。
一次培訓可以容納多少人？
建議每場實作工作坊 20-40 人最佳。演講型課程可支援 200+ 人。透過我們的互動教學簡報系統，所有人都能用手機即時參與互動。
培訓之後會有後續支援嗎？
是的。我們提供培訓後 30 天免費線上諮詢，AI Lobster 方案則包含持續的顧問服務、季度回顧與策略調整。
Claude API 怎麼計費？
API 按 token 計費。Opus 4.6 為 $5/$25 per 1M tokens（Input/Output），Sonnet 為 $3/$15，Haiku 為 $1/$5。可透過 Prompt Caching 和 Batch API 進一步降低成本。
Claude 可以分析 PDF 和圖片嗎？
可以。Claude 支援 PDF 直接上傳分析、圖片辨識（Vision）、截圖解讀等。1M context window 可以一次處理超過 1,000 頁的文件。
什麼是 Constitutional AI？
這是 Anthropic 的核心安全技術，透過一組「AI 憲法」原則來約束模型行為，確保輸出安全、誠實且無害。詳見 Anthropic Research。
企業應該用哪個模型？
建議多模型搭配：Opus 處理複雜分析/策略規劃，Sonnet 處理日常辦公/程式開發，Haiku 用於即時客服/分類任務。根據場景選擇，最佳化成本效益。
什麼是 AI Lobster？
AI Lobster 是我們的企業 AI 生態系導入服務，整合 Claude、ChatGPT、Gemini 等多平台策略，提供從需求評估、工具選型到持續優化的一站式服務。
培訓費用怎麼算？
依企業規模和需求客製報價。單場工作坊約 NT$30,000-80,000，企業全面導入方案另行評估。歡迎填寫表單取得免費報價。
可以只訓練特定部門嗎？
可以。我們提供分部門客製化培訓，可針對行銷、HR、工程、法務等特定部門設計專屬課程內容與實作案例。
有線上課程可以自學嗎？
有的。我們在 tbr.digital 提供 AI 公開課程，涵蓋 ChatGPT、Claude、Gemini 等工具的線上教學，適合個人自學或小團隊學習。
